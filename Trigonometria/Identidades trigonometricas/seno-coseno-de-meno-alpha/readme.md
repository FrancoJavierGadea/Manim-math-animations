## Seno, Coseno, Tangente de menos alpha

$$
\begin{array}{c}
    \sin(-\alpha) = -\sin(\alpha)
    \\\\
    \cos(-\alpha) = \cos(\alpha)
    \\\\
    \tan(-\alpha) = -\tan(\alpha)
\end{array}
$$
